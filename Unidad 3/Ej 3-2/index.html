Ejercicio Practicos N° 3-2

1) Definición de Algoritmo (dar un ejemplo)

Un algoritmo es un conjunto de acciones o pasos finitos, ordenados de forma lógica y que se utilizan para resolver un problema o para obtener un resultado.
Un algoritmo puede ser representado gráficamente mediante un diagrama de flujo.

2) Que es Pseudocódigo (dar un ejemplo)

Se dice que un programa o algoritmo está en pseudocódigo cuando se describen los pasos a seguir en lenguaje "humano".
Ejemplo:
Definir las variables x1 y x2
Ingresar un valor
Asignar el valor ingresado a la variable x1
Ingresar un valor
asignar el valor a la variable x2
calcular la suma de x1+x2
mostrar por consola el resultado de la suma
Para que un algoritmo en pseudocódigo pueda ser interpretado por una computadora, debe ser traducido a algún lenguaje de máquina.

3) Realizar los siguientes problemas:

superficie de un rectángulo
superficie de un triangulo
